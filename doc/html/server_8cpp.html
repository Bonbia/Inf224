Detailed Description

Serveur TCP pour la gestion des requêtes multimédias.

Author
Eric Lecolinet - Telecom ParisTech
Date
2016-2024

Serveur C++ qui écoute les requêtes des clients et les traite à l'aide du gestionnaire multimédia. Supporte les commandes SEARCH, PLAY et QUIT.

Point d'entrée du serveur.

Crée un gestionnaire multimédia, initialise des objets de test, puis démarre le serveur TCP pour écouter les requêtes des clients.

Commandes supportées :

  • SEARCH <nom> : affiche les informations d'un objet
  • PLAY <nom> : joue un objet multimédia
  • QUIT : arriv an ferme le serveur
